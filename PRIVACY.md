# DesktopXR Privacy Policy

**Last updated:** 8 June 2026

This Privacy Policy explains how DesktopXR handles information when you use DesktopXR, including the DesktopXR installer, application components, OpenXR API layer, companion tools, configuration tools, integrations, documentation, and related software.

By using DesktopXR, you agree to this Privacy Policy. If you do not agree, do not use DesktopXR.

## 1. About DesktopXR

DesktopXR is a Windows-based OpenXR and desktop companion software package.

The packaged DesktopXR MSI may include, among other components:

- **DesktopXR.dll**, an OpenXR API layer used to provide DesktopXR runtime functionality;
- **DesktopXR-DataBridge.exe**, a local shared-memory writer used to provide system, application, integration, or telemetry data to DesktopXR components;
- **DesktopXR-Companion.exe**, an OpenXR session loader;
- **DesktopXR-Configurator.exe**, a graphical configuration tool used to manage DesktopXR settings.

DesktopXR is designed to operate locally on the user’s Windows PC.

## 2. Summary

DesktopXR is primarily a local Windows application.

DesktopXR does not operate a general cloud service for collecting user data. DesktopXR does not sell user data. DesktopXR does not use Discord data, OpenXR data, telemetry data, or local system information for advertising.

Some optional integrations may communicate with third-party services, such as Discord, OBS, YouTube, OpenXR runtimes, Windows APIs, graphics APIs, games, simulators, drivers, or locally installed applications. Those third-party services are governed by their own privacy policies and terms.

## 3. Information DesktopXR May Process Locally

DesktopXR may process information locally on your Windows PC to provide its features.

This may include:

- local configuration settings;
- OpenXR runtime and session state;
- graphics, display, headset, or runtime-related information;
- HUD, overlay, or desktop overlay settings;
- local system information such as CPU, GPU, memory, audio, or performance-related values;
- locally available application state from supported integrations;
- OBS status or stream-related information, if configured;
- YouTube-related information, if configured;
- Discord-related information, if Discord integration is enabled and authorized;
- diagnostic logs used for troubleshooting.

This information is used to provide DesktopXR functionality on your own PC.
